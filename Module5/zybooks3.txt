16.1 Function templates
- Multiple functions may be nearly identical, differing only in their data types, as below.
- Writing and maintaining redundant functions that only differ by data type can be time-consuming and error-prone. The language supports a better approach.

- A function template is a function definition having a special type parameter that may be used in place of types in the function
- The function type return type is preceded by template<typename TheType>, where TheType can be any identifier.
- That type is kown as a type perameter and can be sued throughout the function for any parameter types, return types or local variables types.
- The identifier is known as a template parameter, and may be various reference types or even another template parameter.
- The compiler automatically generates a unique function definition for each type appearing in function calls to the function template
